traffic_light.ino 
/*
  Semáforo Inteligente
  Acende LEDs em sequência para simular um semáforo.
*/

// Definição dos pinos
const int PIN_VERMELHO = 13;
const int PIN_AMARELO = 12;
const int PIN_VERDE = 11;

// Tempos em milissegundos
const int TEMPO_VERDE = 5000;
const int TEMPO_AMARELO = 2000;
const int TEMPO_VERMELHO = 5000;

void setup() {
  // Configura os pinos como saída
  pinMode(PIN_VERMELHO, OUTPUT);
  pinMode(PIN_AMARELO, OUTPUT);
  pinMode(PIN_VERDE, OUTPUT);
}

void loop() {
  // 1. Sinal Verde
  digitalWrite(PIN_VERDE, HIGH);
  digitalWrite(PIN_AMARELO, LOW);
  digitalWrite(PIN_VERMELHO, LOW);
  delay(TEMPO_VERDE);

  // 2. Sinal Amarelo
  digitalWrite(PIN_VERDE, LOW);
  digitalWrite(PIN_AMARELO, HIGH);
  delay(TEMPO_AMARELO);

  // 3. Sinal Vermelho
  digitalWrite(PIN_AMARELO, LOW);
  digitalWrite(PIN_VERMELHO, HIGH);
  delay(TEMPO_VERMELHO);
}

Como funciona?

1

Configuração (Setup)

No bloco setup(), definimos os pinos 11, 12 e 13 como SAÍDAS de energia, pois eles vão alimentar os LEDs.

2

O Loop Infinito

A função loop() roda para sempre. Ela liga um LED, espera um tempo com delay(), desliga e passa para o próximo.

3

Desafio para o Aluno

Tente adicionar um "botão de pedestre" que, quando pressionado, força o sinal a ficar vermelho para os carros.
